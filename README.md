# 🎬 RecoFlix - Movie Recommendation System

RecoFlix is a content-based movie recommendation system that suggests similar movies based on user input.

## 🚀 Features
- 🔍 Search any movie
- 🎯 Top 3 "Must Watch" recommendations
- 🎬 Additional similar movies
- ⭐ Ratings display
- 🎭 Category-based suggestions

## 🛠 Tech Stack
- Python
- Pandas
- Scikit-learn
- Flask
- HTML, CSS

## 📊 Model
- Content-Based Filtering
- Cosine Similarity

## 💡 Future Improvements
- Movie posters (TMDB API)
- Better UI/UX
- User login system
